# Privacy Policy for Fandom Spoiler Guard

**Last Updated:** December 6, 2025

## Overview

Fandom Spoiler Guard is a browser extension that protects users from spoilers on Fandom wiki pages. We are committed to protecting your privacy.

## Data Collection

### What We Collect
This extension stores the following data locally using Chrome's storage API:
- **Custom Keywords:** Words you add to identify spoiler content (e.g., "Died", "Status")
- **Hidden Fields:** Infobox fields you manually choose to hide

### What We Do NOT Collect
- Personal information (name, email, etc.)
- Browsing history
- Analytics or usage data
- Any data from the websites you visit

## Data Storage

All data is stored locally on your device using `chrome.storage.sync`. This means:
- Data syncs across your Chrome browsers if you're signed into Chrome
- Data never leaves Google's infrastructure
- We have no access to your data
- No data is sent to external servers

## Third Parties

This extension does not share any data with third parties. We do not use analytics, tracking, or advertising services.

## Data Deletion

You can delete all stored data by:
1. Removing all custom keywords in the extension popup
2. Uninstalling the extension (removes all data)

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last Updated" date above.

## Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
